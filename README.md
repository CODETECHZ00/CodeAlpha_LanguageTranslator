# Language Translation Tool - CodeAlpha Internship

## Project Overview
A web-based language translation application built with Python Flask and Google Translate API. Users can translate text between 100+ languages with an intuitive and modern interface.

## Features
- Translate text between 100+ languages
- Auto-detect source language
- Swap languages with one click
- Copy translated text to clipboard
- Character counter
- Responsive design

## Technologies Used
- Python 3.x
- Flask (Web Framework)
- Google Translate API (googletrans library)
- HTML, CSS, JavaScript

## How to Run
1. Install dependencies:
   ```
   pip install flask googletrans==4.0.0rc1
   ```
2. Run the application:
   ```
   python app.py
   ```
3. Open browser and go to: `http://127.0.0.1:5000`
